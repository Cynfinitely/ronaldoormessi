# Ronaldo V Messi

This project aims to find answer the question of century. Who is the better : Ronaldo or Messi? You can support you idol by clicking button under the name of it. It has got global counter!

## Used techs

1- React
2- TypeScript
3- Tailwind CSS
4- Redux Toolkit
